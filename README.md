# Sup-Event Fullstack Login Example

## Folder Structure

- `backend/`: Node.js Express backend
- `frontend/`: React frontend
- `README.md`: Project documentation

## Steps to Run

1. **Backend**
   - `cd backend`
   - `npm install`
   - `node index.js`

2. **Frontend**
   - `cd frontend`
   - `npm install`
   - `npm start`

Make sure both are running. The frontend is served at `localhost:3000` and backend at `localhost:4000`.
